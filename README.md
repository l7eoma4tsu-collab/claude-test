# 自己紹介サイト

シンプルな自己紹介Webサイトのサンプルプロジェクトです。

## ファイル構成

```
portfolio/
├── index.html   # メインページ
├── style.css    # デザイン・レイアウト
├── script.js    # アニメーション・インタラクション
└── README.md    # このファイル
```

## 掲載内容

- **About Me** — 名前・自己紹介文
- **趣味** — プログラミング / 読書 / ゲーム / サイクリング
- **好きな食べ物** — お寿司 / ラーメン / ピザ / チーズケーキ

## 機能

- レスポンシブデザイン（スマホ対応）
- スクロール連動のフェードインアニメーション
- スムーズスクロールナビゲーション
- スクロール位置に応じたナビのアクティブハイライト
- トップへ戻るボタン

## 使い方

`index.html` をブラウザで開くだけで動作します。サーバー不要。

```
portfolio/index.html をダブルクリック → ブラウザで開く
```

## カスタマイズ

| 変更したい内容 | 編集ファイル |
|---|---|
| 名前・自己紹介・趣味・食べ物の内容 | `index.html` |
| 色・フォント・レイアウト | `style.css` |
| アニメーションの動き | `script.js` |
